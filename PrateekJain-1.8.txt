Steps to run code:

1.Download code from url "https://github.com/jain-prateek/Highest--Company-Share-Price.git" by clicking on button "Download ZIP".

2.Unzip the file "Highest--Company-Share-Price-master".

3.Copy whole foler to D:/ drive.
  >> d:
4.Open Command Prompt and then navigate to "D:\Highest--Company-Share-Price".
  >> cd Highest--Company-Share-Price

5.To get list for each Company year and month in which the share price was highest, run command :
  >> python highest_share_price.py

6.For running unittest to test our program run command :
  >> python test_highestshareprice.py
